# Hi, I’m Emeka 👋

I am an atmospheric scientist and data science enthusiast with strong interests in **Python, machine learning, air quality modeling, remote sensing, and environmental data analytics**.

My work focuses on using data-driven methods to understand atmospheric pollution, improve air quality prediction, and develop reproducible workflows for scientific and applied research.

## 🔬 Current Research and Projects

- 🚀 **Air Pollution Prediction**
  
  I am currently developing deep learning models, including **LSTM-based approaches**, to predict air pollutant concentrations and support improved air quality modeling.

- 🌍 **Atmospheric Science and Black Carbon Research**
  
  My recent collaboration led to a peer-reviewed publication titled:

  **“City-produced and transported black carbon: Synergy of in-situ optical measurements and modeling.”**

  The article is open access and available here:  
  https://doi.org/10.1016/j.atmosres.2025.108731

- 🤖 **Machine Learning Applications**
  
  I have completed projects in:
  
  - Sentiment analysis using modern NLP models
  - Healthcare cost prediction using machine learning
  - Loan default prediction using an Artificial Neural Network implemented with Python, TensorFlow, and Keras

## 🧠 Areas of Interest
